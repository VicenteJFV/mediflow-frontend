# MediFlow - Frontend (Portal Paciente)

Este repositorio contiene la interfaz de usuario del Portal de Pacientes de MediFlow, implementada con React y Vite.

## Arquitectura y Patrones
Se ha implementado el patrón de diseño **Container / Presentational Components** para separar la lógica de obtención de datos de la interfaz gráfica.

## Instrucciones de Instalación y Ejecución

1. Instalar dependencias:
   ```bash
   npm install

2. Iniciar el servidor de desarrollo: 
   ```bash
   npm run dev

El proyecto estará disponible en http://localhost:5173.
